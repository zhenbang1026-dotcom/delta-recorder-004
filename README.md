# 三角洲录制器005

从 `三角洲录制器004` **完整复制** 后的独立动作路线分支。

> **旧项目代码未修改。** 所有改动只在本目录。

## 相对 004 增加的功能

| 功能 | 说明 |
|------|------|
| **Q 动作菜单** | 录制中按 Q 暂停坐标采样；完成/取消后恢复录制和游戏焦点，可连续添加多个动作 |
| **键盘动作** | 单键或 `ctrl+shift+f` 组合键，单击/长按，持续时间单位为毫秒 |
| **等待与注释** | 毫秒等待、中文注释写入 JSONL 和日志 |
| **视角动作** | 记录水平视角，回放时最多 5 次恢复，误差不超过 3° |
| **低头/抬头** | 分步平滑 Y 位移，每步 X 在可配置范围内随机摆动，最后自动补偿 X |
| **YOLO 交互** | `best.onnx` 检测区域固定为 `504,358,952,614`；DirectML 优先、CPU 自动回退；识别失败跳过动作 |
| **YOLO 状态窗口** | 执行 YOLO 动作时弹出实时窗口，显示恢复视角、检测数量、置信度、对准误差、剩余时间和带框截图；窗口不抢游戏焦点 |
| **路线编辑** | JSONL 坐标只读，动作可增删改、上下移动；旧 TXT 编辑时另存 JSONL，不覆盖原文件 |

普通路线不加载 YOLO 模型，低配置电脑（包括 GTX 750）会自动使用 CPU 推理；高配置电脑可使用 DirectML。

## 相对旧版保留的功能

| 优化 | 说明 |
|------|------|
| **稳定截图** | `截图模块.py`：默认 GDI → mss → PIL，不 sleep 重试 |
| **并集一截两裁** | 小地图+角度一次截图，内存裁切 |
| **寻路暂停 UI 识别** | 巡航中 UI 只显示缓存，不抢控制环 |
| **日志缓冲** | 普通 step 批量写盘，关键事件立即落盘 |
| **角度两模式** | **Legacy 丝滑版**（旧颜色轮廓）+ **原版 TEXT**（原 HSV、连通域与 atan2+90） |
| **连续控制与倍率** | 8ms 连续滑动；速度倍率可在 `0.5x～3.0x` 间按 `0.1x` 调节 |
| **定位** | SIFT 与旧项目一致 |

### 角度两模式与默认值

首次启动默认使用 **Legacy**；关闭主界面后会保存并在下次启动恢复上次模式：

- **Legacy 丝滑版**：保留旧颜色轮廓识别、ROI、路线与控制参数，作为默认模式。
- **原版 TEXT**：照搬外部脚本实际入口的蓝色 HSV、蓝色校准、连通域与 atan2+90，ROI 为 `34,78,227,271`。

独立角度识别 UI 也只显示这两个选项，切换模式时会自动切换到对应 ROI。

### Legacy/TEXT 连续控制

Legacy 和 TEXT 均使用 8ms 连续视角控制：

- TEXT 固定标定尺 `33.3 px/°`，Legacy 保留原有在线标定。
- 对鼠标转向限速度、限加速度，抑制相邻控制周期反向振荡。
- 主界面速度滑块范围为 `0.5x～3.0x`、步长 `0.1x`，默认 `1.5x`。
- `1.5x` 默认对应最大角速度 `270°/s`、最大角加速度 `1620°/s²`；需要更大滑动幅度时可继续上调。
- 接近路线点且角度差较大时停止前进并原地连续转向，避免边走边转形成绕圈；小角差仍连续前进微调。

## 路线文件与动作

005 新录制路线保存在 `录制结果/` 和 `routes/`，扩展名为 `.jsonl`；每个路线点可携带动作列表。旧 `.txt` 仍可直接回放。选中旧 TXT 点击“编辑动作”并保存时，会在同目录生成同名 `.jsonl`，原 TXT 保留。

动作按列表顺序执行。YOLO 动作默认首次 F 500ms、首次等待 300ms、循环 F 50ms；W 持续时间、循环 F 次数和间隔均可在 Q 菜单修改，W 时长不足时不允许保存。

YOLO 状态窗口关闭后仍会继续执行动作；窗口状态为“视角恢复失败”“识别/对准超时”或“检测器不可用”时，该动作会跳过，后续路线继续回放。日志中的 `yolo_loaded` 表示模型已加载，`yolo_inference` 表示已实际截图推理，`yolo_aligned` 表示已对准并开始 F/W 交互。

## 合并主界面（主界面.py）

| 功能 | 说明 |
|------|------|
| 开始/停止识别 | 实时坐标+角度 |
| 开始/停止录制 | 防跳变抽稀，保存 JSONL 到 `录制结果/` 与 `routes/` |
| 路线列表 | 自动汇总 TXT/JSONL 的 `routes/` + `录制结果/` |
| Q 动作菜单 | 录制中暂停并编辑动作；保存/取消均从当前位置继续 |
| 路线动作编辑 | “编辑动作”打开已保存路线；坐标只读，旧 TXT 另存 JSONL |
| 开始/停止回放 | 调用旧 `巡航()`，Esc 紧急停止 |
| 角度模式 | Legacy 丝滑版 / 原版 TEXT（内部值 `legacy / text`）；首次启动默认 Legacy |
| 速度倍率 | `0.5x～3.0x`，步长 `0.1x`，默认 `1.5x` |
| 自动保存 | 关闭时保存角度模式、速度倍率、到点阈值、精准模式和所选路线 |

开始识别或回放时，主界面会先最小化并等待 3 秒再启动；停止或任务结束后自动恢复窗口。

`主界面.py` 统一编排两个模式；独立角度识别入口也提供相同选择，首次启动默认值均为 Legacy。
