# 三角洲录制器004

从 `changhe/services/navigation/legacy_delta` **完整复制** 后的独立优化版。

> **旧项目代码未修改。** 所有改动只在本目录。

## 相对旧版做了什么（仅 004）

| 优化 | 说明 |
|------|------|
| **稳定截图** | `截图模块.py`：默认 GDI → mss → PIL，不 sleep 重试 |
| **并集一截两裁** | 小地图+角度一次截图，内存裁切 |
| **寻路暂停 UI 识别** | 巡航中 UI 只显示缓存，不抢控制环 |
| **日志缓冲** | 普通 step 批量写盘，关键事件立即落盘 |
| **角度两模式** | **Legacy 丝滑版**（旧颜色轮廓）+ **原版 TEXT**（原 HSV、连通域与 atan2+90） |
| **连续控制与倍率** | 8ms 连续滑动；速度倍率可在 `0.5x～3.0x` 间按 `0.1x` 调节 |
| **定位** | SIFT 与旧项目一致 |

### 角度两模式与默认值

首次启动默认使用 **Legacy**；关闭主界面后会保存并在下次启动恢复上次模式：

- **Legacy 丝滑版**：保留旧颜色轮廓识别、ROI、路线与控制参数，作为默认模式。
- **原版 TEXT**：照搬外部脚本实际入口的蓝色 HSV、蓝色校准、连通域与 atan2+90，ROI 为 `34,78,227,271`。

独立角度识别 UI 也只显示这两个选项，切换模式时会自动切换到对应 ROI。

### Legacy/TEXT 连续控制

Legacy 和 TEXT 均使用 8ms 连续视角控制：

- TEXT 固定标定尺 `33.3 px/°`，Legacy 保留原有在线标定。
- 对鼠标转向限速度、限加速度，抑制相邻控制周期反向振荡。
- 主界面速度滑块范围为 `0.5x～3.0x`、步长 `0.1x`，默认 `1.5x`。
- `1.5x` 默认对应最大角速度 `270°/s`、最大角加速度 `1620°/s²`；需要更大滑动幅度时可继续上调。
- 接近路线点且角度差较大时停止前进并原地连续转向，避免边走边转形成绕圈；小角差仍连续前进微调。

## 合并主界面（主界面.py）

| 功能 | 说明 |
|------|------|
| 开始/停止识别 | 实时坐标+角度 |
| 开始/停止录制 | 防跳变抽稀，保存到 `录制结果/` 与 `routes/` |
| 路线列表 | 自动汇总 `routes/` + `录制结果/` |
| 开始/停止回放 | 调用旧 `巡航()`，Esc 紧急停止 |
| 角度模式 | Legacy 丝滑版 / 原版 TEXT（内部值 `legacy / text`）；首次启动默认 Legacy |
| 速度倍率 | `0.5x～3.0x`，步长 `0.1x`，默认 `1.5x` |
| 自动保存 | 关闭时保存角度模式、速度倍率、到点阈值、精准模式和所选路线 |

开始识别或回放时，主界面会先最小化并等待 3 秒再启动；停止或任务结束后自动恢复窗口。

`主界面.py` 统一编排两个模式；独立角度识别入口也提供相同选择，首次启动默认值均为 Legacy。

## 与 003 的区别

- **004**：默认提供 Legacy 丝滑版，并保留原版 TEXT atan2+90，方便两模式对比
- **003**：另有 text 角度模式切换等实验能力
