# 三角洲录制器004

从 `changhe/services/navigation/legacy_delta` **完整复制** 后的独立优化版。

> **旧项目代码未修改。** 所有改动只在本目录。

## 相对旧版做了什么（仅 004）

| 优化 | 说明 |
|------|------|
| **稳定截图** | `截图模块.py`：默认 GDI → mss → PIL，不 sleep 重试 |
| **并集一截两裁** | 小地图+角度一次截图，内存裁切 |
| **寻路暂停 UI 识别** | 巡航中 UI 只显示缓存，不抢控制环 |
| **日志缓冲** | 普通 step 批量写盘，关键事件立即落盘 |
| **角度两模式** | **Legacy 丝滑版**（旧颜色轮廓）+ **原版 TEXT**（atan2+90）；**截图方式不变** |
| **双模式控制优化** | Legacy/TEXT 均使用连续限速、限加速度与近点保护；Legacy 提高滑动幅度（见下） |
| **定位** | SIFT 与旧项目一致 |

### 角度两模式与默认值

程序每次启动都默认使用 **Legacy**：

- **Legacy 丝滑版**：保留旧颜色轮廓识别、ROI、路线与控制参数，作为默认模式。
- **原版 TEXT**：使用原版 atan2+90 雷达箭头识别。

### Legacy/TEXT 连续控制

Legacy 和 TEXT 均使用 8ms 连续视角控制：

- TEXT 固定标定尺 `33.3 px/°`，Legacy 保留原有在线标定。
- 对鼠标转向限速度、限加速度，抑制相邻控制周期反向振荡。
- Legacy 最大角速度提高到 `180°/s`、最大角加速度提高到 `1080°/s²`，滑动幅度更大且保持连续。
- 接近路线点且角度差较大时停止前进并原地连续转向，避免边走边转形成绕圈；小角差仍连续前进微调。

## 运行

```powershell

python -m venv .venv
.\.venv\Scripts\pip install -r requirements.txt

.\.venv\Scripts\python 主界面.py
```

```powershell

.\.venv\Scripts\python 自动录制坐标工具.py
```

可选环境变量：`DELTA_CAPTURE_BACKEND=gdi|mss|pil`

## 合并主界面（主界面.py）

| 功能 | 说明 |
|------|------|
| 开始/停止识别 | 实时坐标+角度 |
| 开始/停止录制 | 防跳变抽稀，保存到 `录制结果/` 与 `routes/` |
| 路线列表 | 自动汇总 `routes/` + `录制结果/` |
| 开始/停止回放 | 调用旧 `巡航()`，Esc 紧急停止 |
| 角度模式 | legacy / text；默认 legacy（截图方式不变） |

`主界面.py` 统一编排两个模式；独立录制入口也提供相同选择，启动默认值均为 Legacy。

## 与 003 的区别

- **004**：默认提供 Legacy 丝滑版，并保留原版 TEXT atan2+90，方便两模式对比
- **003**：另有 text 角度模式切换等实验能力
