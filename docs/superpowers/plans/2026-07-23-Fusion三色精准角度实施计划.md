# Fusion 三色精准角度实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在不修改 Legacy 和现有 TEXT 识别行为的前提下，为 Fusion 增加绿色、蓝色、黄色独立标定的精准主观测、几何门禁和两帧颜色切换确认。

**Architecture:** 新建 Fusion 专用 `三色精准角度.py`，组合现有 `识别角度.角度识别器` 与三份独立 MAP 标定。`A测试角度识别._analyze_image_fusion` 把精准结果放入现有双源选择器的主观测槽，Legacy 继续同帧校验和降级；颜色识别器第二帧提供观测，但最终输出仍服从 Fusion 一致性选择。

**Tech Stack:** Python 3、OpenCV、NumPy、pytest、dataclass、现有 MAP 圆形残差插值。

---

## 文件职责

- `三色精准角度.py`：三色 profile、标定校验、几何门禁、质量评分、颜色锁定状态机；不截图、不控制鼠标。
- `校准数据/Fusion/{绿色,蓝色,黄色}/lv.txt`：Fusion 专用运行时标定，仅三份文本。
- `A测试角度识别.py`：只增加 Fusion 精准识别器的惰性加载、状态重置、主观测转换和诊断映射。
- `tests/test_fusion_three_color_profiles.py`：profile、资产指纹、标定合法性和 MAP 插值质量。
- `tests/test_fusion_three_color_recognition.py`：三色合成图、门禁、质量分和两帧切换。
- `tests/test_angle_mode_regression.py`：冻结 Legacy/TEXT，验证 Fusion 接入和模式隔离。
- `README.md`：说明 Fusion 三色自动识别、两帧确认和 Legacy 降级。

```python
def _函数源码哈希(path: Path, name: str) -> str:
    source = path.read_text(encoding="utf-8")
    tree = ast.parse(source)
    node = next(
        item for item in tree.body
        if isinstance(item, (ast.FunctionDef, ast.AsyncFunctionDef)) and item.name == name
    )
    text = "\n".join(source.splitlines()[node.lineno - 1:node.end_lineno]) + "\n"
    return hashlib.sha256(text.encode("utf-8")).hexdigest()
```

```python
center_score = clamp(1.0 - center_error / 5.0)

quality = 0.35 * center_score + 0.30 * radius_score + 0.20 * mask_score + 0.15 * map_confidence
```
